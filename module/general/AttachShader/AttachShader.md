AttachShader
============
attach GLSL shader for use in COVER

Input ports
-----------
|name|description|
|-|-|
|data_in||

Output ports
------------
|name|description|
|-|-|
|data_out||

Parameters
----------
|name|type|description|
|-|-|-|
|shader|String|name of shader to apply to geometry|
|shader_params|String|shader parameters (as "key=value" "key=value1 value2"|
