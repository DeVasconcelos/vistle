
# AttachShader
Attach GLSL shader for use in COVER

## Input ports
|name|description|
|-|-|
|data_in||


## Output ports
|name|description|
|-|-|
|data_out||


## Parameters
|name|description|type|
|-|-|-|
|shader|name of shader to apply to geometry|String|
|shader_params|shader parameters (as "key=value" "key=value1 value2"|String|
