
# ColorAttribute
Specify solid object color as an attribute

## Input ports
|name|description|
|-|-|
|data_in|input data|


## Output ports
|name|description|
|-|-|
|data_out|output data|


## Parameters
|name|description|type|
|-|-|-|
|color|hexadecimal RGB/RGBA values (#rrggbb or #rrggbbaa)|String|
