
# TransformGrid
Shuffle and mirror axes of rectilinear grids

## Input ports
|name|description|
|-|-|
|data_in|input data|


## Output ports
|name|description|
|-|-|
|data_out|output data|


## Parameters
|name|description|type|
|-|-|-|
|reverse X-axix||Integer|
|reverse Y-axix||Integer|
|reverse Z-axix||Integer|
|orderOfGridAxes| (XYZ, XZY, YXZ, YZX, ZXY, ZYX)|Integer|
